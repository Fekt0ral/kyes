KYES (know your expenses) - веб-приложение по просмотру своих трат в месяц/год на подписки на различные сервисы.

Бизнес-логика: 
- отправка уведомлений пользователю за выбранное им число дней до оплаты подписки с целью понимания необходимости оставления или продления подписки
- сумма трат в месяц, год на подписки
- фильтры по категориям (только развлечения или развлечения и ИИ)

CRUD:
- добавление - название, дата оплаты, раз в месяц/год/другой период, сумма (руб, дол, евро), категория
- изменение
- просмотр ближайших оплат по категориям
- удаление

БД:
- User (id, name, email, password_hash)
- Subscription (id, user_id, service_name, price, currency, payment_date, category, cycle)
